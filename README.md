# Clash-Verge-Global-Extension-Script
用于分流，使网站访问速度更快/可访问。

## Feature
分组包括AI和社交媒体
- AI分组使用新加坡/日本/美国等地的节点（香港等地容易被block）
- 社交媒体分组使用香港节点（速度快且没有block）
- 全局自动模式：
  1. 如果机场没有以上地区的节点，则使用延迟最低节点。
  2. 如果分组节点都timeout，也使用全局自动节点。
  3. 如果分组节点恢复（没timeout），从全局自动节点转换为分组节点。
 
## 注意事项
- 为了防止节点频繁变化，tolerance设置为150，即如果分组里，当前使用节点和最快节点延迟差距小于150ms，就不切换。

## 环境 
### 电脑端：clash verge
- 在订阅内找到全局扩展脚本进行修改
  
### 手机端：FLClash
- 写脚本：工具->进阶配置->脚本->修改脚本
- 配置到机场：代理->更多->覆写->覆写模式选择脚本->覆写脚本选择刚才创建的脚本
